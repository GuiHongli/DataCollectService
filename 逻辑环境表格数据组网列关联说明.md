## 数据关联实现

### 1. 后台服务层更新

#### LogicEnvironmentServiceImpl 更新
- **数据源**: 使用 `NetworkTypeService` 和 `NetworkType` 实体
- **查询方法**: `getNetworksByIds()` 方法直接查询 `network_type` 表
- **数据转换**: `createNetworkInfoList()` 方法将网络类型数据转换为前端需要的格式

```java
private List<NetworkType> getNetworksByIds(List<Long> networkIds) {
    QueryWrapper<NetworkType> networkQuery = new QueryWrapper<>();
    networkQuery.in("id", networkIds);
    return networkTypeService.list(networkQuery);
}
```

#### 数据转换流程
```java
private List<LogicEnvironmentDTO.NetworkInfo> buildNetworkInfoList(LogicEnvironment logicEnvironment) {
    // 1. 获取关联的网络ID列表
    List<Long> networkIds = getNetworkIdsByEnvironment(logicEnvironment.getId());
    
    // 2. 根据网络ID查询网络类型数据表
    List<NetworkType> networks = getNetworksByIds(networkIds);
    
    // 3. 转换为前端需要的格式
    return createNetworkInfoList(networks);
}
```

### 3. 数据表关联关系

#### 核心数据表
- **主表**: `network_type` (网络类型数据表)
- **关联表**: `logic_environment_network` (逻辑环境与网络关联表)
- **查询逻辑**: 先查关联表获取网络ID，再查网络类型表获取详细信息

#### 关联查询实现
```java
// 1. 查询关联表获取网络ID列表
private List<Long> getNetworkIdsByEnvironment(Long environmentId) {
    QueryWrapper<LogicEnvironmentNetwork> networkQueryWrapper = new QueryWrapper<>();
    networkQueryWrapper.eq("logic_environment_id", environmentId);
    List<LogicEnvironmentNetwork> logicEnvironmentNetworks = logicEnvironmentNetworkService.list(networkQueryWrapper);
    return logicEnvironmentNetworks.stream()
        .map(LogicEnvironmentNetwork::getLogicNetworkId)
        .collect(Collectors.toList());
}

// 2. 根据网络ID查询网络类型表
private List<NetworkType> getNetworksByIds(List<Long> networkIds) {
    QueryWrapper<NetworkType> networkQuery = new QueryWrapper<>();
    networkQuery.in("id", networkIds);
    return networkTypeService.list(networkQuery);
}
```

#### 实时同步
- 网络类型数据的任何修改都会立即反映到表格数据中

#### 状态管理
- 通过 `network_type.status` 字段控制网络类型的可见性

### 2. 数据转换

#### DTO转换
```java
private List<LogicEnvironmentDTO.NetworkInfo> createNetworkInfoList(List<NetworkType> networks) {
    List<LogicEnvironmentDTO.NetworkInfo> networkInfoList = new ArrayList<>();
    for (NetworkType network : networks) {
        LogicEnvironmentDTO.NetworkInfo networkInfo = new LogicEnvironmentDTO.NetworkInfo();
        networkInfo.setId(network.getId());
        networkInfo.setName(network.getName());
        networkInfo.setDescription(network.getDescription());
        networkInfoList.add(networkInfo);
    }
    return networkInfoList;
}
```
