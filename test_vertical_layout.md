# 采集任务创建页面竖向布局测试

## 布局变更说明

### 原布局（横向步骤）
- 4个步骤横向排列
- 需要点击"下一步"按钮切换步骤
- 每次只显示一个步骤的内容

### 新布局（竖向步骤）
- 4个步骤竖向排列
- 所有步骤同时显示在一个页面上
- 不需要点击"下一步"按钮
- ~~左侧显示步骤指示器，右侧显示所有步骤内容~~
- **直接显示所有步骤内容，无步骤指示器**
