# 用例集 Excel 模板说明

## 文件名要求
- 文件名必须为：`case.xlsx`
- 文件格式：Excel 2007+ (.xlsx)

## 表格结构

### 第一行（表头）
Excel 文件的第一行为表头，包含以下 10 列：

| 列号 | 列名 | 英文字段名 | 说明 |
|-----|------|-----------|------|
| 1 | 用例_名称 | name | 测试用例的名称 |
| 2 | 用例_编号 | number | 测试用例的唯一编号 |
| 3 | 用例_逻辑组网 | logicNetwork | 逻辑组网环境，多个用分号分隔 |
| 4 | 用例_业务大类 | businessCategory | 业务分类 |
| 5 | 用例_APP | app | 应用名称（中文） |
| 6 | 用例_APPEN | appEn | 应用英文名称 |
| 7 | 用例_模型场景 | modelScenario | 模型场景描述 |
| 8 | 用例_手机OS类 | phoneOsType | 手机操作系统类型（如：Android、iOS） |
| 9 | 用例_操作步骤 | testSteps | 详细的测试步骤 |
| 10 | 用例_预期结果 | expectedResult | 预期的测试结果 |

### 数据行
从第二行开始为实际的测试用例数据。

## 字段说明

### 必填字段
- **用例_名称**（第1列）：不能为空
- **用例_编号**（第2列）：不能为空

### 可选字段
其他字段均为可选，可以留空。

### 字段格式说明

#### 1. 用例_逻辑组网
多个逻辑组网用分号（;）分隔，例如：
```
4G标准网络;4G弱网环境
```

#### 2. 用例_操作步骤
可以使用换行符分隔多个步骤，例如：
```
1. 启动测试应用
2. 检查网络连接状态
3. 执行网络连接测试
4. 验证连接稳定性
```

#### 3. 用例_手机OS类
建议使用以下标准值：
- Android
- iOS
- HarmonyOS
- 其他

## 生成测试 Excel

可以使用提供的 Python 脚本生成符合格式的测试 Excel 文件：

```bash
python create_test_excel.py
```

该脚本会在 `test_upload` 目录下生成 `case.xlsx` 文件。

## 注意事项

1. **编码格式**：确保 Excel 文件使用 UTF-8 编码保存
2. **空行处理**：空行会被自动忽略
3. **数据验证**：上传前请确保：
   - 用例名称和编号不为空
   - 用例编号在同一用例集内唯一
4. **特殊字符**：避免在字段中使用特殊控制字符
5. **单元格格式**：建议将所有单元格格式设置为"文本"，避免数字格式转换问题

## 上传流程

1. 准备符合格式的 `case.xlsx` 文件
2. 将 `case.xlsx` 放入 ZIP 压缩包
3. 在用例集管理页面点击"上传用例集"
4. 选择 ZIP 文件上传
5. 系统会自动解析 Excel 文件并导入测试用例

## 常见问题

### Q: 为什么上传失败提示"未找到 case.xlsx"？
A: 确保 ZIP 压缩包中包含名为 `case.xlsx` 的文件，且文件名大小写正确。

### Q: 用例编号可以包含字母吗？
A: 可以，用例编号支持字母、数字及下划线的组合。

### Q: 操作步骤如何换行？
A: 在 Excel 单元格中使用 Alt+Enter 进行手动换行，或使用编号列表格式。

### Q: 如果某些可选字段为空会怎样？
A: 系统会正常导入，前端页面会显示"未配置"提示。
