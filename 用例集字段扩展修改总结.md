# 用例集字段扩展修改总结

## 修改时间

## 修改目标
扩展用例集功能，支持解析和显示 case.xlsx 文件中的完整 10 列数据，新增 APPEN、模型场景、手机OS类 三个字段。

## 兼容性说明

### 向后兼容
- 新增字段均为可选字段（数据库允许 NULL）
- 现有数据不受影响，新字段会显示为"未配置"（中文）/ "Not configured"（英文）
- 旧版 API 请求不受影响
- 支持中英文语言切换，用户体验更友好

### Excel 格式要求
- **新上传**的用例集应使用 10 列格式的 Excel 文件
- 旧格式（7列）的 Excel 文件可能无法正确解析，建议更新为新格式

### 数据迁移
现有数据的新增字段值为 NULL，前端会显示"未配置"。如需补充数据，可以：
1. 重新上传包含完整字段的用例集
2. 或后续提供数据编辑功能进行补充

## 注意事项

1. **数据库备份**: 执行 SQL 升级前务必备份 `test_case` 表
2. **服务停机窗口**: 建议在低峰期进行部署
3. **前后端同步部署**: 前后端需要一起升级，避免版本不匹配
4. **用户通知**: 通知用户更新 Excel 模板，使用新的 10 列格式
5. **回滚方案**: 如遇问题，可回滚到旧版本，新增字段数据不会影响旧版本运行

## 性能影响

- 新增字段为普通 varchar 类型，不会显著影响查询性能
- 表数据量不大时，ALTER TABLE 操作通常在秒级完成
- 前端页面新增 3 列，横向滚动条会出现，已设置用例名称为固定列优化体验
